# dictionary

## What is dictionary?

**dictionary** is a program, which enables you to create and view your own version of English dictionary.

  
## Requirements
gcc v.13.3.0+

## Installation & running

```sh
git clone https://github.com/GeorgiiGlazkov/dictionary 
cd dictionary
make
./dictionary
```

## Usage

- add *word* -- adds new word to the dictionary
- delete *word* -- deletes existing word from the dictionary
- draw -- prints all the words from the dictionary
- find -- checks if the word exists in the dictionary
- help -- see this help message
- exit -- exits the program
